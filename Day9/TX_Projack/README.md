## 海量即时通讯系统

### 界面

- 功能：能够正确

- 需求分析 
    - 1) 用户注册 
    - 2) 用户登录 
    - 3) 显示在线用户列表 
    - 4) 群聊(广播) 
    - 5) 点对点聊天 
    - 6) 离线留言
